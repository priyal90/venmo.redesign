Venmo

Case Study

Overview

Venmo is not a banking App, yet is a payment app. This is a redesign of Venmo to optimize the main mission of the application for the intended user. A close study on mirco interactions to be fun, friendly and social payment system. Send and receive money with friends. The case study was end to end; from discovery, user research, user testing to design and deploy.

How might we optimise the user experience that is fun, friendly and social with a focus feedback loop while transacting money.

User Testing

Think Aloud Cognitive Walkthrough


Task 1. How would you navigate through the app.

Task 2.  Ask users about how often do they check their balance.

Task 3. Ask users about the menu bar.  

Task 4. Ask users about Scan QR Codes.
‍
Task 5. Ask users about the amount slider.

Task 6. Ask users about confirmation feedback.

Task 7. Ask users about the speed of the animation.
‍

Key Insights


1. Users wanted their profile to be more  emphasize.

2. Users like the menu bar rather than a profile image.

3. Users were not using most of the existing touch points.

4. Depending on the age and scenario, users would user the amount slider

5. The speed of animation and feed back mattered a lot.

6. Style guide and colors need to cater Venmo.
‍

Solution


1. Inserted a Dashboard, to display the balance and give an option to transfers funds.

2. Inserted a clear payment confirmation feedback.
‍
3. Inserted mirco interactions making the app Fun and Friendly.
